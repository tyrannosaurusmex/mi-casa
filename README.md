# T-Mex - Mi Cassa

### Requirements

##### Node + NPM

```
https://nodejs.org/en/download/
```

##### Grunt

```
npm install -g grunt-cli
```

##### Scss-Lint

```
gem install scss-lint
```

### Installation

Install node modules:

```
npm install
```

Compile app and run on local server:

```
grunt
```

Visit [http://localhost:3000/](http://localhost:3000/) in your browser
